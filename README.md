# Enterprise Data Quality Monitoring System

## Overview
This project simulates enterprise data governance by applying automated data quality rules on a real-world dataset.

## Features
- Data profiling (missing values, dtype analysis)
- Duplicate detection
- Outlier detection using IQR
- Data quality scoring system

## Tech Stack
Python, Pandas, NumPy

## Dataset
Titanic dataset (public)

## Output
- Data quality score per record
- Cleaned dataset export
